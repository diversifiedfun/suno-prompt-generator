🎧 Design a Set

A whole multi-track playlist with an energy arc.

❔ How this works

Four steps: 1 pick an occasion, 2 set how long, 3 fill in the sentence (and add a ✍️ lyrics theme if you want words), then Build my set. You'll get an energy arc you can nudge (▲/▼) plus a recurring motif; then Generate all tracks and Paste next → Suno one at a time. Every set auto-saves under 📁 My sets. ❔ Getting started & help

Get a key (~2 min): go to platform.claude.com → sign in → add credits (pay-as-you-go, ~a few cents per set) → open platform.claude.com/settings/keys → Create Key → copy the sk-ant-… value → paste above → Save. Stored only in this browser, sent only to Anthropic — don't use a shared computer.

No key? The Vibe and Set tabs fall back to a built-in library (you'll see a note). Build and Library work fully without a key.

Make one song: ✨ Vibe (describe a feeling) or 🎛 Build (pick ingredients). Make a set: 🎧 Set (occasion → length → theme → Build). Paste into Suno: open a suno.com tab, log in, then use “Paste next → Suno”.

Something blank or stuck? Reload the extension at chrome://extensions (↻), reopen the panel. Full walkthrough: docs/USER-GUIDE.md in the project.
