# LOCK-IN — 12-track deep hypnotic flow-state house set

Deep-focus / flow-state house — heavy & dark but **weightless, not furious**. BPM 124→128.
Arc = a journey DEEPER into flow, then a controlled euphoric surfacing (NOT a fist-pump triumph).
Separate from the older "UNDER THE MISTERS" day-party set and the "NO DAYS OFF" grind phonk set.
