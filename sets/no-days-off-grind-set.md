# NO DAYS OFF — motivational / grind phonk set

Phonk / hard-trap **grind & go-hard** set. Aggressive spoken/rap bars (coach-in-your-ear),
distorted 808s + Memphis cowbell + tape-saturated drums, dark and menacing, hype. This is the
AGGRESSIVE register (opposite of the LOCK-IN flow set) — chosen deliberately because Molly asked
for grind/go-hard directly. Full Style + Exclude + structured lyrics written in chat 2026-07-02.

## Locked spec
- **Length target: 3–5 min per track** (Molly's note: LOCK-IN tracks kept shrinking to ~2:30
  because hook-dense choruses have too few sections). Fix baked in: every track = 16-bar
  instrumental intro + 2 verses + breakdown + **THREE drops** (each hook is a drop w/ 808 slam)
  + builds/risers between. Section count is the length lever, not lyric length.
- **More interesting lyrics** (Molly's note): specific + a little clever, NOT gym-bro clichés.
  Signature lines like "you don't rise to the goal, you fall to the habit" / "not angry —
  hungry" / "luck's just the name that the lazy give to bold" / "nobody saw it, that's how I
  know it was real." Approved by Molly.
- **Register: aggressive/menacing**, hard from bar 1; what climbs is menace + weight, not tempo.
- **Vocal: aggressive male rap / spoken bars**, gritted, coach-in-ear. Vary cadence per track.
- **Universal EXCLUDE (every track):** happy pop, cheesy supersaw, soft EDM, tropical house,
  clean polished vocal, mellow, dreamy. (Enforces the register — see dj-set-playbook.)
- BPM ladders 145→155. Arc: decide → creed → the work → the pain → beast peak → cold payoff.
- Ends COLD/satisfied (Last Light On), not a cheesy victory lap — same instinct as LOCK-IN's
  "Still Locked" ending.
